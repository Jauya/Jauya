## Lo que hago

- Desarrollo web full-stack: desde UI responsive hasta backend y APIs.
- Automatización de procesos con herramientas como **n8n** y scripts en Bash, Node.js y Python.
- Gestión de flujos de comunicación usando **WhatsApp Business API**, **Evolution API** y **Chatwoot**.
- Despliegue de aplicaciones en servidores Linux usando Docker, Coolify y Dokploy.
- Migración y gestión de servidores y correos por terminal (SSH, `authorized_keys`, etc.), aprovechando la velocidad y control del uso directo de la terminal.
- SEO técnico y on‑page: optimización de Core Web Vitals, estructura semántica, metadatos, sitemap, Google Search Console y Google Analytics.
- Integraciones con servicios externos: Cloudflare (DNS, seguridad), APIs de pago, email marketing, generación de contenido con IA, etc.
- Gestión de múltiples sitios web en producción, asegurando rendimiento, seguridad y actualizaciones.
